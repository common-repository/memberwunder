Ein WordPress e-Learning (LMS) Plugin, um sogenannte WordPress Learning Management Systeme zu erstellen mit anpassbaren Designs und sofort einsetzbarem Front-End. (Kurse, Lektionen, Quizze und mehr)

== Description ==

**<a href="https://memberwunder.com/" target="_blank">Das MemberWunder LMS Plugin</a>** ist ein perfektes *WordPress Learning Management System Plugin*. Es ist eines der besten WordPress LMS Plugins mit kostenlosen Design-Vorlagen. Diese können benutzt werden, um Kurse simpel anzulegen und diese online zu verkaufen. Jeder Kurs kann mit Modulen, Lektionen und Quizze erstellt werden. Diese wiederum lassen sich leicht per Drag-and-Drop verwalten.

<a href="https://demo.memberwunder.com/" target="_blank">Live Vorschau</a>

MemberWunder LMS ist kostenlos und wird immer kostenlos sein, jedoch bieten wir ein hochqualitatives MemberWunder Plugin an. Dieses wird Ihnen dabei helfen mit ihrem **MemberWunder LMS** Geld zu verdienen. Versuchen Sie es und sehen Sie selbst.


###MemberWunder LMS Funktionen###

**MemberWunder LMS - Ein WordPress LMS Plugin mit fertigem Design**
Wir bieten zum MemberWunder LMS Plugin 7 verschiedene Designs an und obendrauf sogar die Möglichkeit ein selbst-angefertigtes Design zu nutzen.

**MemberWunder LMS unterstützt WordPress Multisite**
Erstellen Sie sogar WordPress basierte LMS’ in Form von WordPress Multisite.

**Kurse erstellen**
Das MemberWunder LMS Plugin bietet Ihnen eine exzellente Benutzeroberfläche um Ihre Kurse so anzulegen, wie Sie es wünschen.
Sie können sogar bestehende Kurse exportieren und in andere Webseiten einpflegen, die das MemberWunder LMS Plugin ebenfalls benutzen.

**Kurse verwalten**
Sie können Ihren Kurs teilen, verwalten und vieles mehr!

**Kurse verkaufen**
MemberWunder LMS Pro erlaubt es Ihnen Ihre Kurse mit Zahlungsanbietern wie Digistore24 zu verknüpfen.

**LearnPress wird ständig weiterentwickelt**
Wir entwickeln und verbessern MemberWunder LMS tagtäglich und bringen Ihnen neue Funktionen mit dem Ziel, MemberWunder LMS zum **besten WordPress LMS Plugin** zu machen.

== Installation ==

**Über Ihr WordPress Dashboard**
1. Gehen Sie zu 'Plugin > Neues hinzufügen'.
2. Suchen Sie nach 'LearnPress'.
3. Aktivieren Sie MemberWunder LMS auf Ihrer Plugins Seite.

**Über WordPress.org**
1. Suchen und laden Sie MemberWunder LMS herunter.
2. Aktivieren Sie das Plugin über das “Plugin” Menü im WordPress Dashboard

Oder

1. Laden Sie ihren Plugin Ordner in das ‘wp-content/plugins’ Verzeichnis
2. Aktivieren Sie das Plugin über das “Plugin” Menü im WordPress Dashboard
